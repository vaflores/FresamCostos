# Fresam Context v1.0

## Visión
Sistema Integral para Control de Costos de Fresam.

## Stack Definitivo
Backend: ASP.NET Core 8, Dapper, SQL Server 2022, JWT, xUnit
Frontend: Angular 19, Angular Material, Reactive Forms
Hosting: IIS

- CRUD Create/Read Seguridad
- JWT
- Claims
- PerfilSeguridad
- Policies dinámicas
- IIS
- CORS
- Angular
- Login Responsive
- Angular Material
